## Project Description
Smart Study Web App is a productivity web application for students.
Users can manage tasks, notes, timer sessions, and music.
The app also includes dark/light mode and animated background.

## Technologies
- Frontend: Angular
- Backend: Django REST Framework
- Database: SQLite/PostgreSQL
- Authentication: JWT

## Features
- Login / Logout
- Task CRUD
- Notes CRUD
- Timer
- Music
- Dark/Light mode
- Animated background
